# {{projectName}}

Subscription platform on Arc Network (Substack/Patreon alternative), built with ArcPay.

## Features
- Monthly/yearly USDC subscriptions
- Auto-prorated refund on cancel
- 2% protocol fee (vs 10% on Patreon/Substack)
- Global — works for creators Stripe doesn't support

## Quick start
```bash
npm install && npm run dev
```

Edit `.env.local` for config, then deploy with `npx vercel`.
